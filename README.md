# ETL Pipeline using Python and MySQL

## Objective
Build a simple ETL pipeline using Python and MySQL.
## Steps
1. Extract data from CSV
2. Clean missing values using Pandas
3. Transform data
4. Analyze sales data
5. Store cleaned data

## Technologies
- Python
- Pandas
- MySQL
- SQL

## Results
- Generated cleaned dataset
- Calculated total sales
- Performed city-wise sales analysis
